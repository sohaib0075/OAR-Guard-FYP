# Dashboard

Interactive clinician-facing dashboard for exploring per-patient OAR-Guard results.

Tabs:
- **Overview** — dose heatmap + top-ranked organs
- **Heatmap Explorer** — dose maps across body locations
- **Ranking Table** — full organ ranking with statistics
- **Downloads** — export generated results

Replace `app.py` with your actual dashboard implementation (e.g. Streamlit, Flask, or Dash) and update the run instructions in the main [README](../README.md#getting-started).
